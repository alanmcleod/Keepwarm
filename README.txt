KEEPWARM

A simple browser-based writing progress tracker.

Files to upload to GitHub Pages:
- index.html
- styles.css
- app.js
- manifest.webmanifest
- service-worker.js
- icon.svg

The app saves its data in the browser on the device being used. It does not yet sync between devices.

Main features:
- Multiple book projects
- Positive and negative word changes
- Project totals
- Words today and yesterday across all projects
- Entry history with running totals
- Rename, archive, restore and delete projects
- Works offline after first opening
- Can be added to an iPhone Home Screen from Safari
